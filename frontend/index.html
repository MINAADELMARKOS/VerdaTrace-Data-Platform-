VerdaTrace Data Platform

VerdaTrace — A GCP-native platform for trusted, governed, and sustainable data pipelines.

Cloud Run URL: replace after deployment with https://SERVICE-REGION-PROJECT.run.app

Ingest

Upload customer files, publish streaming events, or run Kaggle demo loaders through Pub/Sub and Cloud Storage.

Govern

Apply IAM, Cloud DLP inspection, Secret Manager salts, KMS encryption, BigQuery retention and Dataplex metadata.

Detect

Flag high fare-per-mile mobility spend, high transport emissions, duplicate records and raw direct identifiers.

Operate

Use Cloud Logging, Monitoring alerts, audit logs and dead-letter topics to run the platform with production controls.

Platform capabilities

  • Mobility expense assurance: detects suspicious trip costs and tip ratios.
  • ESG transport emissions: estimates CO2e and flags high-emission activity.
  • Retail privacy pipeline: pseudonymises customer IDs and flags direct identifiers before analytics use.
  • Backend processing: GKE worker consumes Pub/Sub and writes curated BigQuery records.

Demo command

gcloud run deploy verdatrace-portal --source frontend --region europe-west2 --allow-unauthenticated
